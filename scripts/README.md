# 📜 스크립트 폴더

이 폴더에는 프로젝트 실행 및 유틸리티 스크립트가 포함되어 있습니다.

## 📋 포함된 파일

- `demo.py` - 데모 실행 스크립트
- `view_dashboard.py` - 대시보드 뷰어
- `run_dashboard.sh` - 대시보드 실행 스크립트
- `build_debug.spec` - 디버깅용 빌드 설정 (콘솔 모드)

## 🚀 사용 방법

### 데모 실행
```bash
python scripts/demo.py
```

### 대시보드 뷰어 실행
```bash
python scripts/view_dashboard.py
```

### 대시보드 실행 스크립트
```bash
./scripts/run_dashboard.sh
```

---

**참고**: 메인 애플리케이션은 프로젝트 루트에서 실행하세요.
- 데스크톱 앱: `python main.py`
- 웹 앱: `python app.py`
