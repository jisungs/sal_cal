## ✅ 구현 내용

### 1. 세션 타임아웃 설정 추가

**파일**: `config.py`

**변경 사항**:
```python
# 세션 설정
SESSION_COOKIE_SECURE = False  # HTTPS 사용 시 True로 변경
SESSION_COOKIE_HTTPONLY = True
SESSION_COOKIE_SAMESITE = 'Lax'
PERMANENT_SESSION_LIFETIME = 86400  # 24시간 (초 단위)
```

**설명**:
- `PERMANENT_SESSION_LIFETIME`: 세션 유지 시간 (86400초 = 24시간)
- Flask-Login과 함께 작동하여 세션 만료 관리

### 2. 세션 쿠키 설정 확인

**현재 설정**:
- `SESSION_COOKIE_SECURE = False` (개발 환경)
- `SESSION_COOKIE_HTTPONLY = True` (XSS 방지)
- `SESSION_COOKIE_SAMESITE = 'Lax'` (CSRF 방지)

**프로덕션 환경**:
- `SESSION_COOKIE_SECURE = True` (HTTPS 필수)
