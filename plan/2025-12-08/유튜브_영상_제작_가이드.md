# 🎬 유튜브 영상 제작 가이드

**목적**: MVP 프로젝트 홍보 및 수익화를 위한 유튜브 영상 제작 가이드  
**대상**: 개발자, 기술 블로거, 교육 콘텐츠 제작자

---

## 📋 영상 제작 전략

### 영상 1: MVP 데모 영상 (2-3분) ⭐ 필수
**목표**: 빠른 시연으로 관심 유도

**구성:**
1. **문제 소개** (10초)
   - "급여명세서 작성이 번거로우신가요?"
   - 화면: 급여명세서 작성하는 모습

2. **솔루션 소개** (20초)
   - "10초만에 자동 생성하는 웹 서비스"
   - 화면: 웹 서비스 메인 화면

3. **데모** (1-2분)
   - 엑셀 업로드 → 자동 계산 → PDF 다운로드
   - 화면: 실제 사용 과정 녹화

4. **CTA** (10초)
   - "무료로 사용해보세요" + 링크
   - 화면: 링크 및 연락처

**촬영 팁:**
- 화면 녹화 소프트웨어 사용 (OBS Studio, QuickTime)
- 마우스 움직임 부드럽게
- 불필요한 대기 시간 편집으로 제거

---

### 영상 2: 개발 과정 영상 (10-15분) ⭐ 수익화
**목표**: 교육 콘텐츠 + 패키지 판매

**구성:**
1. **프로젝트 소개** (1분)
   - 프로젝트 목표 및 기능 소개
   - 화면: 프로젝트 구조

2. **개발 과정** (8-10분)
   - Flask 웹 앱 구조 설명
   - 파일 업로드 구현 과정
   - PDF 생성 구현 과정
   - 화면: 코드 및 실행 화면

3. **코드 공개** (2-3분)
   - 핵심 코드 설명
   - 화면: 코드 하이라이트

4. **패키지 판매** (1분)
   - "전체 코드 + 템플릿 패키지: 39,000원"
   - 화면: 구매 링크 및 연락처

**촬영 팁:**
- 타임랩스 활용 (코딩 과정)
- 화면 녹화 + 음성 설명
- 코드 하이라이트 도구 사용

---

### 영상 3: 사용법 가이드 (5-7분) ⭐ 선택
**목표**: 사용자 온보딩

**구성:**
1. **서비스 소개** (30초)
   - 서비스 기능 소개
   - 화면: 서비스 메인 화면

2. **사용법 설명** (4-5분)
   - 엑셀 파일 준비 방법
   - 업로드 방법
   - 결과 다운로드 방법
   - 화면: 단계별 화면 녹화

3. **팁 및 주의사항** (1분)
   - 자주 하는 실수
   - 유용한 팁
   - 화면: 주의사항 텍스트

4. **CTA** (30초)
   - "무료로 사용해보세요" + 링크
   - 화면: 링크 및 연락처

**촬영 팁:**
- 사용자 관점에서 설명
- 실제 사용 시나리오 활용
- 자막으로 단계별 설명

## 📊 업로드 최적화

### 제목 작성 팁
- 키워드 포함: "급여명세서", "자동화", "웹 서비스"
- 숫자 포함: "10초만에", "1주일 완성"
- 감정어 포함: "쉽게", "빠르게"

**예시:**
- "급여명세서 10초만에 자동 생성하는 웹 서비스 만들기"
- "1주일만에 만든 급여명세서 자동화 도구 (무료 코드 공개)"

### 설명 작성 팁
1. **첫 3줄이 중요** (미리보기)
   - 핵심 내용 요약
   - 링크 포함

2. **타임스탬프 추가**
   - 주요 섹션 타임스탬프
   - 예: "0:30 - 데모 시작"

3. **해시태그 활용**
   - #급여명세서 #자동화 #웹개발 #파이썬

### 태그 전략
- 핵심 키워드: 급여명세서, 자동화, 웹서비스
- 기술 키워드: Flask, Python, 웹개발
- 타겟 키워드: 인사팀, 회계, 소상공인
