# 🎨 Day 1: 화면 설계서 (와이어프레임)

## 📋 개요

급여명세서 자동생성기의 GUI 화면 설계 문서입니다.  
**GUI 프레임워크**: tkinter (Python 내장)  
**화면 크기**: 900x750px (고정 크기)

---

## 🖥️ 전체 화면 구조

### 탭 구조
- **탭 1**: 📊 대시보드 (메인 화면)
- **탭 2**: 📄 급여관리 (급여입력, 급여대장, 급여명세서 생성)
- **탭 3**: 📈 급여현황조회 (P3 - 향후 구현)
- **탭 4**: ⚙️ 기초정보 설정 (P0 - 향후 구현)

### 구성 요소

#### 1. 주요 지표 카드

- **데이터**: 직원별 실수령액
- **라이브러리**: matplotlib
- **색상**: #304FFE (Deep Blue)

#### 3. 근무자구성 그래프
- **타입**: 도넛 그래프 (Pie Chart with hole)
- **데이터**: 정규직/계약직/신입 비율
- **라이브러리**: matplotlib
- **색상**: #304FFE, #FFD54F, #4CAF50

#### 4. 특이사항 영역
- 연장근무 있는 직원 수
- 상여금 지급 직원 수

### 구성 요소

#### 1. 입력 파일 영역
- **파일 경로 입력**: 텍스트 입력 필드
- **찾아보기 버튼**: 파일 선택 다이얼로그
- **인라인 오류 메시지**: 빨간색 텍스트로 표시
  - 예: "입력 형식에 문제가 있어요! 😊 파일을 확인해주세요."

#### 2. 파일 미리보기 영역
- **컴포넌트**: ttk.Treeview
- **컬럼**: 이름, 주민번호(마스킹), 입사일, 부양가족수, 기본급
- **행 수**: 최대 3행
- **주민번호 마스킹**: `123456-*******` 형식

#### 3. 설정 영역
- **생성 기간**: YYYY-MM 형식 (예: 2025-01)
- **출력 형식**: 라디오 버튼
  - 엑셀
  - PDF
  - 둘 다 (기본값)
- **출력 방식**: 라디오 버튼
  - 개별 출력 (직원별 개별 파일)
  - 일괄 출력 (전체 직원 통합 파일) (기본값)

#### 4. 출력 폴더 영역
- **폴더 경로 입력**: 텍스트 입력 필드
- **찾아보기 버튼**: 폴더 선택 다이얼로그
- **기본값**: `./output`

#### 5. 액션 버튼
- **생성하기**: 급여명세서 생성 시작
- **초기화**: 모든 입력 필드 초기화

#### 6. 진행 상태 영역
- **진행률 바**: ttk.Progressbar (0-100%)
- **상태 텍스트**: 현재 처리 중인 직원명 및 진행률
  - 예: "처리 중: 홍길동 (3/10)"

#### 7. 완료 후 액션
- **출력 폴더 열기 버튼**: 완료 후 활성화
- **플랫폼별 동작**:
  - Windows: `os.startfile()`
  - macOS: `subprocess.Popen(['open', folder_path])`
  - Linux: `subprocess.Popen(['xdg-open', folder_path])`
