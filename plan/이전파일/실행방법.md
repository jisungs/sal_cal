# 🚀 급여명세서 자동생성기 실행 방법

**버전**: v1.0  
**작성일**: 2025-11-12

---

## 📋 실행 파일 위치

생성된 실행 파일은 다음 위치에 있습니다:
- **macOS**: `dist/급여명세서생성기.app`
- **Windows**: `dist\급여명세서생성기.exe`

---

## 🍎 macOS 실행 방법

### 방법 1: Finder에서 실행 (가장 간단)

1. **Finder 열기**
2. **프로젝트 폴더로 이동**
   - 경로: `/Users/jisungs/Documents/dev/sideprojects/salary_cal`
3. **`dist` 폴더 열기**
4. **`급여명세서생성기.app` 더블클릭**

### 방법 2: 터미널에서 실행

```bash
# 프로젝트 폴더로 이동
cd /Users/jisungs/Documents/dev/sideprojects/salary_cal

# .app 번들 실행
open dist/급여명세서생성기.app
```

### 방법 3: Dock에 추가하여 실행

1. **Finder에서 `dist/급여명세서생성기.app` 찾기**
2. **Dock으로 드래그**
3. **Dock에서 클릭하여 실행**

---

## 🪟 Windows 실행 방법

### 방법 1: 파일 탐색기에서 실행

1. **파일 탐색기 열기**
2. **프로젝트 폴더로 이동**
3. **`dist` 폴더 열기**
4. **`급여명세서생성기.exe` 더블클릭**

### 방법 2: 명령 프롬프트에서 실행

```cmd
cd C:\path\to\salary_cal
dist\급여명세서생성기.exe
```

## ⚠️ macOS 보안 경고 해결 방법

첫 실행 시 macOS 보안 경고가 나타날 수 있습니다:

### "급여명세서생성기.app을 열 수 없습니다" 오류

**해결 방법**:

1. **시스템 환경설정 열기**
2. **보안 및 개인 정보 보호** 선택
3. **일반** 탭 확인
4. **"급여명세서생성기.app"이 차단되었다는 메시지 확인**
5. **"확인" 버튼 클릭** 또는 **"허용" 버튼 클릭**

또는:

1. **Finder에서 `급여명세서생성기.app` 우클릭**
2. **"열기" 선택**
3. **경고 창에서 "열기" 버튼 클릭**

## 📝 실행 후 첫 단계

1. **직원 정보 파일 선택**
   - 대시보드 탭에서 "📁 파일 선택" 버튼 클릭
   - 또는 메뉴바 > 파일 > 직원 정보 파일 열기

2. **템플릿 파일 사용**
   - `payroll_generator/templates/employee_template.xlsx` 파일 사용 가능

3. **급여명세서 생성**
   - 급여명세서 생성 탭에서 파일 선택
   - 출력 형식 선택 (엑셀/PDF/둘 다)
   - "생성하기" 버튼 클릭

## 🔧 문제 해결

### GUI 창이 열리지 않는 경우

1. **터미널에서 실행하여 오류 확인**:
   ```bash
   ./dist/급여명세서생성기.app/Contents/MacOS/급여명세서생성기
   ```

2. **로그 파일 확인**:
   - 위치: `~/.급여명세서생성기/logs/app.log`

3. **콘솔 모드로 재빌드**:
   - `build_mac.spec`에서 `console=True`로 변경
   - 재빌드 후 오류 메시지 확인

### 프로그램이 즉시 종료되는 경우

1. **로그 파일 확인**
2. **필수 파일 확인**:
   - 템플릿 파일: `templates/employee_template.xlsx`
   - 폰트 파일: `assets/NanumGothic.ttf`
