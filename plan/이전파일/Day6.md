## 📝 PyInstaller 빌드 설정 상세

### build.spec 파일 구조

```python
# build.spec
# -*- mode: python ; coding: utf-8 -*-

block_cipher = None

a = Analysis(
    ['main.py'],
    pathex=[],
    binaries=[],
    datas=[
        ('payroll_generator/templates', 'templates'),
        ('payroll_generator/assets/NanumGothic.ttf', 'assets'),
    ],
    hiddenimports=[
        'matplotlib',
        'matplotlib.backends.backend_tkagg',
        'pandas',
        'openpyxl',
        'reportlab',
        'PIL',
    ],
    hookspath=[],
    hooksconfig={},
    runtime_hooks=[],
    excludes=[],
    win_no_prefer_redirects=False,
    win_private_assemblies=False,
    cipher=block_cipher,
    noarchive=False,
)

pyz = PYZ(a.pure, a.zipped_data, cipher=block_cipher)

exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.zipfiles,
    a.datas,
    [],
    name='급여명세서생성기',
    debug=False,
    bootloader_ignore_signals=False,
    strip=False,
    upx=True,
    upx_exclude=[],
    runtime_tmpdir=None,
    console=False,  # GUI 모드
    disable_windowed_traceback=False,
    argv_emulation=False,
    target_arch=None,
    codesign_identity=None,
    entitlements_file=None,
    icon='payroll_generator/assets/icon.ico',  # 아이콘 파일 (선택)
)
```

### 빌드 시 주의사항

1. **폰트 파일 포함**: NanumGothic.ttf 파일이 반드시 포함되어야 함
2. **템플릿 파일 포함**: templates 폴더의 모든 파일 포함
3. **숨겨진 import**: matplotlib, pandas 등 숨겨진 import 명시
4. **파일 크기**: 목표 50MB 이하
5. **테스트**: 빌드 후 반드시 테스트 필요
