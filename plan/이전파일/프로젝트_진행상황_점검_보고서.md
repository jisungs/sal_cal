## 🐛 해결한 주요 문제

### 1. PyInstaller 실행 파일 문제 ✅
- **문제**: 실행 파일이 생성되었지만 실행되지 않음
- **원인**: 리소스 경로 문제, 설정 파일 경로 문제
- **해결**: `resource_path()` 함수 추가, 사용자 홈 디렉토리에 설정 파일 저장

### 2. GUI 창 표시 문제 ✅
- **문제**: macOS에서 GUI 창이 표시되지 않음
- **원인**: matplotlib 모듈 누락, 백엔드 설정 문제
- **해결**: `hiddenimports`에 matplotlib 추가, 백엔드 명시적 설정

### 3. 앱 아이콘 실행 문제 ✅
- **문제**: `.app` 번들이 폴더로 보임
- **원인**: macOS Finder 설정 또는 권한 문제
- **해결**: 권한 설정 확인, `Info.plist` 검증

### 4. Segmentation Fault 문제 ✅ (2025-11-12)
- **문제**: `python main.py` 실행 시 segmentation fault 발생
- **원인**: `setup_matplotlib_font()`에서 GUI 초기화 전에 `plt.figure()` 호출
- **해결**: 폰트 테스트 로직 제거, `rcParams`만 설정

### 5. 플랫폼별 폰트 설정 문제 ✅
- **문제**: macOS 전용 폰트 설정으로 인한 플랫폼 호환성 문제
- **원인**: `plt.rcParams['font.family'] = 'AppleGothic'` 하드코딩
- **해결**: 플랫폼별 폰트 자동 선택 로직 구현
