# 📐 와이어프레임 분석 문서

## 📋 와이어프레임 구조 분석

### 전체 레이아웃
와이어프레임은 **2x2 그리드 레이아웃**으로 구성되어 있습니다.

```
┌─────────────────────────────────────────────┐
│              대시보드 (제목)                 │
├─────────────────────────────────────────────┤
│  ┌──────────────┐  ┌─────────────────────┐ │
│  │ 직원수       │  │                     │ │
│  │ 총급여       │  │                     │ │
│  │ 총공제       │  │                     │ │
│  └──────────────┘  └─────────────────────┘ │
├─────────────────────────────────────────────┤
│  ┌──────────────┐  ┌─────────────────────┐ │
│  │ 월급여 정산  │  │ 근무현황            │ │
│  │ 현황         │  │ - 정규직            │ │
│  │ (막대 그래프)│  │ - 계약직            │ │
│  │              │  │ - 신입              │ │
│  │ 1월 2월 3월  │  │                     │ │
│  └──────────────┘  └─────────────────────┘ │
├─────────────────────────────────────────────┤
│  ┌──────────────┐  ┌─────────────────────┐ │
│  │ 근무자 구성  │  │ 특기사항            │ │
│  │ (파이 차트)  │  │                     │ │
│  │              │  │ [엑셀 파일 불러오기]│ │
│  └──────────────┘  └─────────────────────┘ │
└─────────────────────────────────────────────┘
```

---

## 🔍 상세 구성 요소 분석

### 1. 상단 영역
- **제목**: "대시보드" (우측 상단)
- **직원 정보 카드** (좌측 상단)
  - 직원수
  - 총급여
  - 총공제

### 2. 중간 영역 (2열)

#### 왼쪽: 월급여 정산현황 (막대 그래프)
- **타입**: 세로 막대 그래프
- **X축**: 월별 (1월, 2월, 3월, 4월)
- **Y축**: 급여 금액
- **특징**: 
  - 각 월마다 여러 개의 막대가 있음 (직원별 또는 카테고리별)
  - 1월의 막대가 가장 높음

#### 오른쪽: 근무현황 (텍스트 리스트)
- 정규직: [값]
- 계약직: [값]
- 신입: [값]
- 하단에 요약: "○정규 ○계약 ○신임"

### 3. 하단 영역 (2열)

#### 왼쪽: 근무자 구성 (파이 차트)
- **타입**: 파이 차트 (원형 그래프)
- **범례**:
  - ○ 정규직
  - • 계약직
  - • 신입
- **특징**: 3개 구간으로 나뉘어져 있음

#### 오른쪽: 특기사항 (특이사항)
- 빈 텍스트 영역
- 하단에 **"엑셀 파일 불러오기"** 버튼

## 🔄 현재 구현 vs 와이어프레임 비교

### ✅ 일치하는 부분
1. ✅ 직원수, 총급여, 총공제 표시
2. ✅ 막대 그래프 (급여 현황)
3. ✅ 파이 차트 (근무자 구성)
4. ✅ 특이사항 영역
5. ✅ 엑셀 파일 불러오기 기능

### ❌ 차이점 및 개선 필요 사항

#### 1. 레이아웃 구조
- **와이어프레임**: 2x2 그리드 레이아웃 (모든 요소가 한 화면에)
- **현재 구현**: 탭 구조 (급여 현황 / 근무자구성 탭으로 분리)
- **개선**: 와이어프레임대로 2x2 그리드로 변경 필요

#### 2. 막대 그래프 데이터
- **와이어프레임**: 월별 데이터 (1월, 2월, 3월, 4월)
- **현재 구현**: 직원별 데이터
- **개선**: 월별 급여 추이를 보여주는 그래프 추가 필요
  - 단, 현재는 월별 데이터가 없으므로 임시로 직원별 표시 유지
  - 향후 월별 데이터 수집 기능 추가 필요

#### 3. 근무현황 표시 방식
- **와이어프레임**: 텍스트 리스트 형태 (정규직, 계약직, 신입)
- **현재 구현**: 특이사항 영역에 포함되어 있음
- **개선**: 별도 영역으로 분리하여 표시

#### 4. 총 실수령액 표시
- **와이어프레임**: 명시적으로 보이지 않음 (총급여, 총공제만 표시)
- **현재 구현**: 총 실수령액도 표시
- **개선**: 와이어프레임에 맞춰 총 실수령액을 별도로 강조 표시

#### 5. 버튼 위치
- **와이어프레임**: 특이사항 영역 하단
- **현재 구현**: 상단 우측
- **개선**: 와이어프레임대로 특이사항 영역 하단으로 이동

## 📝 개선 제안

### 우선순위 1: 레이아웃 재구성
1. 탭 구조 제거 → 2x2 그리드 레이아웃으로 변경
2. 모든 그래프와 정보를 한 화면에 표시

### 우선순위 2: 근무현황 영역 분리
1. 근무현황을 별도 영역으로 분리
2. 정규직/계약직/신입 수치를 명확히 표시

### 우선순위 3: 월별 데이터 지원 (향후)
1. 월별 급여 데이터 수집 기능 추가
2. 월별 급여 추이 그래프 구현

### 우선순위 4: UI 요소 위치 조정
1. 엑셀 파일 불러오기 버튼을 특이사항 영역 하단으로 이동
2. 총 실수령액을 별도 카드로 강조 표시
