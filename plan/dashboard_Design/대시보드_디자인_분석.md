## 🎨 디자인 요소 분석

### 1. 색상 팔레트

#### 카드 색상
- **총 직원 수 카드**: 연한 파란색 (Light Blue) - #E3F2FD 또는 유사
- **근무현황 카드**: 진한 파란색 (Dark Blue) - #1976D2 또는 유사
- **특이사항 카드**: 주황색 (Orange) - #FF9800 또는 유사

#### 그래프 색상
- **막대 그래프**:
  - 정규직: 연한 파란색
  - 계약직: 진한 파란색
- **도넛 차트**:
  - 정규직: 연한 파란색 (62.5%)
  - 계약직: 중간 파란색 (25%)
  - 신입: 진한 파란색 (12.5%)

### 2. 상단 카드 영역 (3개 카드)

#### 카드 1: 총 직원 수 (연한 파란색)
- **제목**: "총 직원 수"
- **내용**:
  - 총 직원 수: 10명
  - 총급여: 500만원
  - 총공제: 59만원
- **특징**: 
  - 카드 형태 (둥근 모서리)
  - 배경색: 연한 파란색
  - 텍스트: 검정색

#### 카드 2: 근무현황 (진한 파란색)
- **제목**: "근무현황"
- **내용**:
  - 정규직: 10명
  - 계약직: 20명
  - 신입: 2명
- **특징**:
  - 카드 형태 (둥근 모서리)
  - 배경색: 진한 파란색
  - 텍스트: 흰색 또는 밝은 색

#### 카드 3: 특이사항 (주황색)
- **제목**: "특이사항 :"
- **내용**: 비어있음 (플레이스홀더)
- **특징**:
  - 카드 형태 (둥근 모서리)
  - 배경색: 주황색
  - 텍스트: 검정색 또는 어두운 색

### 3. 하단 차트 영역 (2개 차트)

#### 차트 1: 월별 근무자 수 추이 (막대 그래프)
- **타입**: 클러스터 막대 그래프 (Clustered Bar Chart)
- **범례**:
  - 정규직 (연한 파란색 원)
  - 계약 (진한 파란색 원)
- **X축**: 월별 (1월, 2월, 3월, 4월, 5월, 6월)
- **Y축**: 인원 수 (0~20, 5단위)
- **데이터**:
  - 1월: 정규직 ~3명, 계약직 ~6명
  - 2월: 정규직 ~8명, 계약직 ~14명
  - 3월: 정규직 ~16명, 계약직 ~18명
  - 4월: 정규직 ~14명, 계약직 ~14명
  - 5월: 정규직 ~12명, 계약직 ~12명
  - 6월: 정규직 ~15명, 계약직 ~16명
- **특징**: 
  - 두 개의 막대가 나란히 표시 (클러스터)
  - 월별 추이를 한눈에 파악 가능

#### 차트 2: 근무자 구성 (도넛 차트)
- **타입**: 도넛 차트 (Donut Chart)
- **데이터**:
  - 정규직: 62.5% (연한 파란색)
  - 계약직: 25% (중간 파란색)
  - 신입: 12.5% (진한 파란색)
- **특징**:
  - 중앙에 구멍이 있는 원형 차트
  - 비율을 시각적으로 표현
  - 색상으로 구분

## 🔄 현재 구현 vs 실제 디자인 비교

### ✅ 일치하는 부분
1. ✅ 총 직원 수, 총급여, 총공제 표시
2. ✅ 근무현황 (정규직, 계약직, 신입) 표시
3. ✅ 특이사항 영역
4. ✅ 도넛 차트 (근무자 구성)
5. ✅ 막대 그래프

### ❌ 차이점 및 개선 필요 사항

#### 1. 레이아웃 구조
- **실제 디자인**: 상단 3개 카드 (가로 배치) + 하단 2개 차트 (가로 배치)
- **현재 구현**: 왼쪽 지표/특이사항 + 오른쪽 탭 구조 (그래프)
- **개선**: 실제 디자인대로 3개 카드 + 2개 차트 레이아웃으로 변경

#### 2. 카드 디자인
- **실제 디자인**: 
  - 3개의 색상이 다른 카드 (연한 파란색, 진한 파란색, 주황색)
  - 둥근 모서리
  - 카드별 배경색과 텍스트 색상 대비
- **현재 구현**: 
  - 단색 배경의 LabelFrame
  - 색상 구분 없음
- **개선**: 
  - 카드 스타일 적용 (둥근 모서리, 배경색)
  - 각 카드별 색상 구분

#### 3. 막대 그래프 데이터 ✅
- **실제 디자인**: 
  - 월별 데이터 (1월~6월)
  - 클러스터 막대 그래프 (정규직/계약직 나란히)
  - 6개월 추이 데이터
- **현재 구현**: 
  - ✅ 월별 데이터 생성 (임시 - 현재 데이터 기반 추정)
  - ✅ 클러스터 막대 그래프 구현 완료
  - ✅ 세로 막대 그래프 (X축: 월, Y축: 인원 수)
- **개선 완료**: 
  - ✅ 월별 클러스터 막대 그래프 구현
  - ⏳ 실제 월별 이력 데이터 수집 기능 추가 (향후)

#### 4. 총 실수령액 표시
- **실제 디자인**: 총급여, 총공제만 표시 (총 실수령액은 명시적 표시 없음)
- **현재 구현**: 총 실수령액도 표시
- **개선**: 실제 디자인에 맞춰 총 실수령액 제거 또는 별도 표시

#### 5. 특이사항 영역
- **실제 디자인**: 
  - 주황색 카드
  - 비어있는 상태로 표시
  - 제목: "특이사항 :"
- **현재 구현**: 
  - 흰색 배경
  - 텍스트 영역에 내용 표시
- **개선**: 주황색 카드 스타일 적용

#### 6. 파일 선택 버튼
- **실제 디자인**: 버튼이 보이지 않음 (상단에 있을 것으로 추정)
- **현재 구현**: 상단 우측에 "📁 파일 선택" 버튼
- **개선**: 버튼 위치 유지 (디자인에 명시되지 않았으므로)

## 📝 개선 제안

### 우선순위 1: 레이아웃 재구성
1. **상단**: 3개 카드를 가로로 배치
   - 카드 1: 총 직원 수 (연한 파란색)
   - 카드 2: 근무현황 (진한 파란색)
   - 카드 3: 특이사항 (주황색)

2. **하단**: 2개 차트를 가로로 배치
   - 차트 1: 월별 근무자 수 추이 (막대 그래프)
   - 차트 2: 근무자 구성 (도넛 차트)

### 우선순위 2: 카드 디자인 적용
1. **카드 스타일**:
   - 둥근 모서리 (border-radius)
   - 배경색 적용
   - 그림자 효과 (선택사항)
   - 적절한 패딩

2. **색상 적용**:
   - 총 직원 수: 연한 파란색 (#E3F2FD)
   - 근무현황: 진한 파란색 (#1976D2, 텍스트 흰색)
   - 특이사항: 주황색 (#FF9800)

### 우선순위 3: 그래프 개선
1. **막대 그래프**:
   - 클러스터 막대 그래프로 변경
   - 정규직/계약직을 나란히 표시
   - 월별 데이터 지원 (향후)

2. **도넛 차트**:
   - 현재 구현 유지 (이미 도넛 차트)
   - 색상 조정 (연한 파란색, 중간 파란색, 진한 파란색)

### 우선순위 4: 데이터 표시 개선
1. **총 실수령액**: 
   - 실제 디자인에 명시되지 않았으므로 제거 또는 별도 표시
   - 또는 총급여 - 총공제로 계산 가능하도록 유지

2. **특이사항**:
   - 주황색 카드 스타일 적용
   - 비어있을 때도 카드 형태 유지
