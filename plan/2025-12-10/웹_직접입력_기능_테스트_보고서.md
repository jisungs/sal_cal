## 📝 테스트 실행 방법

### 1. 웹 서버 실행
```bash
cd /Users/jisungs/Documents/dev/sideprojects/salary_cal
source venv/bin/activate
python app.py
```

### 2. 브라우저에서 접속
- 메인 페이지: http://localhost:5001/
- 직접 입력 페이지: http://localhost:5001/input/

### 3. 테스트 시나리오
1. 네비게이션 메뉴에서 "직접 입력" 클릭
2. 필수 정보 입력:
   - 이름: 홍길동
   - 주민번호 앞자리: 123456
   - 입사일: 2024-01-01
   - 기본급: 3000000
   - 부양가족수: 1
3. 선택 정보 입력 (선택사항):
   - 연장근무시간: 10
   - 연장근무단가: 25000
   - 상여금: 500000
4. 실시간 미리보기 확인
5. 지급 기간 입력: 2025-01
6. 출력 형식 선택: 둘 다
7. "급여명세서 생성" 버튼 클릭
8. 결과 페이지에서 파일 다운로드 확인
