# 📁 샘플 파일 가이드

**작성일**: 2025-11-11  
**프로젝트**: 급여명세서 자동생성기 v1.0

---

## 📋 개요

이 문서는 급여명세서 자동생성기에서 제공하는 샘플 파일에 대한 설명입니다.

---

## 📂 샘플 파일 위치

### 입력 샘플 파일

- **위치**: `payroll_generator/templates/`
- **파일명**: 
  - `employee_template.xlsx`: 직원 정보 템플릿
  - `company_template.xlsx`: 회사 정보 템플릿 (선택사항)

### 출력 샘플 파일

- **위치**: `payroll_generator/output/`
- **파일명**:
  - `생성된_급여명세서_샘플.xlsx`: 엑셀 출력 샘플
  - `생성된_급여명세서_샘플.pdf`: PDF 출력 샘플

---

## 📝 입력 샘플 파일 설명

### employee_template.xlsx

직원 정보를 입력하는 템플릿 파일입니다.

#### 포함된 데이터

| 이름 | 주민번호 | 입사일 | 부양가족수 | 기본급 | 연장근무시간 | 연장근무단가 | 상여금 |
|------|----------|--------|------------|--------|--------------|--------------|--------|
| 홍길동 | 123456-1 | 2024-01-01 | 1 | 3000000 | 10 | 25000 | 500000 |
| 김영희 | 234567-2 | 2024-02-01 | 2 | 3500000 | 5 | 30000 | 0 |
| 이철수 | 345678-1 | 2024-03-01 | 0 | 2800000 | 0 | 0 | 0 |

#### 사용 방법

1. 템플릿 파일을 엑셀에서 엽니다.
2. 샘플 데이터를 참고하여 실제 직원 정보를 입력합니다.
3. 필수 컬럼은 반드시 입력해야 합니다.
4. 파일을 저장합니다.
5. 프로그램에서 이 파일을 선택하여 급여명세서를 생성합니다.

---

## 📄 출력 샘플 파일 설명

### 생성된_급여명세서_샘플.xlsx

생성된 엑셀 급여명세서 샘플입니다.

#### 포함된 내용

- **지급 항목**:
  - 기본급
  - 연장근무수당
  - 상여금
  - 총 지급액

- **공제 항목**:
  - 국민연금
  - 건강보험
  - 장기요양
  - 고용보험
  - 소득세
  - 지방소득세
  - 총 공제액

- **최종 항목**:
  - 실수령액

#### 특징

- 전문적인 스타일링 적용
- 금액은 천 단위 구분 기호(,) 포함
- 계산 결과 자동 반영

### 생성된_급여명세서_샘플.pdf

생성된 PDF 급여명세서 샘플입니다.

#### 포함된 내용

- **헤더**: 급여명세서 제목 및 지급 기간
- **직원 정보**: 이름, 주민번호(마스킹)
- **지급 항목 테이블**: 지급 항목 및 금액
- **공제 항목 테이블**: 공제 항목 및 금액
- **실수령액**: 강조 표시

#### 특징

- A4 크기
- 한글 폰트 지원 (나눔고딕)
- 주민번호 자동 마스킹 처리
- 전문적인 레이아웃

---

## 💡 사용 예시

### 예시 1: 템플릿 파일 사용

1. `employee_template.xlsx` 파일을 엑셀에서 엽니다.
2. 샘플 데이터를 참고하여 실제 직원 정보를 입력합니다.
3. 파일을 저장합니다.
4. 프로그램에서 이 파일을 선택하여 급여명세서를 생성합니다.

### 예시 2: 출력 파일 확인

1. 프로그램에서 급여명세서를 생성합니다.
2. 출력 폴더에서 생성된 파일을 확인합니다.
3. `생성된_급여명세서_샘플.xlsx` 또는 `생성된_급여명세서_샘플.pdf`를 참고하여 형식을 확인합니다.

## ⚠️ 주의사항

1. **템플릿 파일 수정**: 템플릿 파일의 컬럼명은 변경하지 마세요.
2. **데이터 형식**: 날짜는 `YYYY-MM-DD` 형식으로 입력하세요.
3. **숫자 형식**: 금액은 숫자만 입력하세요 (천 단위 구분 기호 제외).
4. **주민번호**: 주민번호 앞자리만 입력하세요 (뒷자리는 자동 마스킹).
