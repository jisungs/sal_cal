## 1. 프로그램 소개

### 1.1 프로그램 목적

급여명세서 자동생성기는 소상공인과 소규모 사업주를 위한 간편한 급여 계산 및 명세서 생성 프로그램입니다. 복잡한 4대보험 계산과 세금 계산을 자동화하여 급여 업무 시간을 대폭 단축할 수 있습니다.

### 1.2 주요 기능

- ✅ **4대보험 자동 계산**
  - 국민연금 (4.5%, 상한액 5,530,000원)
  - 건강보험 (3.545%, 상한액 5,530,000원)
  - 장기요양 (건강보험의 12.95%)
  - 고용보험 (0.9%, 상한액 5,530,000원)

- ✅ **소득세/지방소득세 자동 계산**
  - 간이세액표 기반 5단계 누진세율 적용
  - 부양가족수별 공제 반영 (0~4명 이상)

- ✅ **엑셀/PDF 자동 생성**
  - 직원별 개별 급여명세서 생성
  - 엑셀 및 PDF 형식 지원
  - 전문적인 스타일링 적용
  - **디자인 선택 기능** (v1.1 신규)
    - 기본 디자인, 디자인 1, 디자인 2 중 선택 가능
    - 웹 및 데스크톱 인터페이스 모두 지원

- ✅ **대시보드 기능**
  - 직원 수, 총 급여, 총 공제, 총 실수령액 표시
  - 월급여 정산현황 막대 그래프
  - 근무자구성 도넛 그래프
  - 특이사항 표시

### 1.3 타겟 사용자

- 10인 이하 소상공인
- 프리랜서 고용주
- 소규모 학원/과외 운영자
- 카페/식당 사장님
- 기타 소규모 사업주

## 3. 설치 방법

### 3.1 실행 파일 다운로드

1. 프로그램 실행 파일을 다운로드합니다.
   - Windows: `급여명세서생성기.exe`
   - macOS: `급여명세서생성기` (실행 파일)
   - Linux: `급여명세서생성기` (실행 파일)

### 3.2 설치 과정

#### Windows

1. 다운로드한 `급여명세서생성기.exe` 파일을 더블클릭합니다.
2. Windows Defender 또는 바이러스 백신 프로그램에서 경고가 표시될 수 있습니다.
   - 이는 정상적인 현상입니다. "추가 정보" → "실행"을 클릭하세요.
3. 프로그램이 실행됩니다.

#### macOS

1. 다운로드한 `급여명세서생성기` 파일을 원하는 위치에 복사합니다.
2. 터미널에서 다음 명령어를 실행합니다:
   ```bash
   chmod +x 급여명세서생성기
   ./급여명세서생성기
   ```
3. 또는 Finder에서 파일을 우클릭 → "열기"를 선택합니다.
4. 보안 경고가 표시되면 "열기"를 클릭합니다.

#### Linux

1. 다운로드한 `급여명세서생성기` 파일을 원하는 위치에 복사합니다.
2. 터미널에서 다음 명령어를 실행합니다:
   ```bash
   chmod +x 급여명세서생성기
   ./급여명세서생성기
   ```

### 3.3 첫 실행

1. 프로그램을 실행하면 메인 화면이 표시됩니다.
2. 처음 실행 시 대시보드 탭이 기본으로 표시됩니다.
3. 직원 정보 엑셀 파일을 준비하여 사용을 시작할 수 있습니다.

## 4. 엑셀 템플릿 작성 가이드

### 4.1 템플릿 파일 위치

프로그램과 함께 제공되는 템플릿 파일:
- `templates/employee_template.xlsx`: 직원 정보 템플릿
- `templates/company_template.xlsx`: 회사 정보 템플릿 (선택사항)

### 4.2 필수 컬럼 설명

직원 정보 엑셀 파일에는 다음 컬럼이 **반드시** 포함되어야 합니다:

| 컬럼명 | 타입 | 필수 | 설명 | 예시 |
|--------|------|------|------|------|
| 이름 | 문자열 | ✅ | 직원 이름 | 홍길동 |
| 주민번호 | 문자열 | ✅ | 주민번호 앞자리 (세금계산용) | 123456-1 |
| 입사일 | 날짜 | ✅ | 입사일 (YYYY-MM-DD 형식) | 2024-01-01 |
| 부양가족수 | 정수 | ✅ | 부양가족 수 (소득세 계산용) | 1 |
| 기본급 | 정수 | ✅ | 월 기본급 | 3000000 |

### 4.3 선택 컬럼 설명

다음 컬럼은 선택사항입니다:

| 컬럼명 | 타입 | 필수 | 설명 | 예시 |
|--------|------|------|------|------|
| 계좌번호 | 문자열 | ❌ | 급여 이체용 계좌번호 | 123-456-789012 |
| 연장근무시간 | 정수 | ❌ | 월 연장근무 시간 | 10 |
| 연장근무단가 | 정수 | ❌ | 시간당 연장근무 단가 | 25000 |
| 상여금 | 정수 | ❌ | 상여금 금액 | 500000 |
| 공제사항 | 문자열 | ❌ | 기타 공제 메모 | - |

### 4.4 샘플 데이터 작성 방법

1. 템플릿 파일을 엑셀에서 엽니다.
2. 각 직원의 정보를 행 단위로 입력합니다.
3. 필수 컬럼은 반드시 입력해야 합니다.
4. 선택 컬럼은 필요에 따라 입력합니다.

**예시 데이터**:

| 이름 | 주민번호 | 입사일 | 부양가족수 | 기본급 | 연장근무시간 | 연장근무단가 | 상여금 |
|------|----------|--------|------------|--------|--------------|--------------|--------|
| 홍길동 | 123456-1 | 2024-01-01 | 1 | 3000000 | 10 | 25000 | 500000 |
| 김영희 | 234567-2 | 2024-02-01 | 2 | 3500000 | 5 | 30000 | 0 |
| 이철수 | 345678-1 | 2024-03-01 | 0 | 2800000 | 0 | 0 | 0 |

### 4.5 주의사항

- ⚠️ **주민번호**: 주민번호 앞자리만 입력하세요. 뒷자리는 자동으로 마스킹 처리됩니다.
- ⚠️ **날짜 형식**: 입사일은 반드시 `YYYY-MM-DD` 형식으로 입력하세요.
- ⚠️ **숫자 형식**: 기본급, 연장근무시간, 연장근무단가, 상여금은 숫자만 입력하세요.
- ⚠️ **컬럼명**: 컬럼명은 정확히 일치해야 합니다 (대소문자 구분).

## 5. 사용 방법

### 5.1 대시보드 사용법

#### 5.1.1 대시보드 화면 구성

대시보드 탭은 다음과 같이 구성되어 있습니다:

- **상단**: 제목 및 파일 선택 버튼
- **카드 위젯**: 주요 지표 표시
  - 총 직원 수
  - 총 급여
  - 총 공제
  - 총 실수령액
- **그래프**: 시각화 데이터
  - 월급여 정산현황 (막대 그래프)
  - 근무자구성 (도넛 차트)
- **특이사항**: 연장근무, 상여금 등 특이사항 표시

#### 5.1.2 파일 선택 및 업데이트

1. **"📁 파일 선택"** 버튼을 클릭합니다.
2. 직원 정보 엑셀 파일을 선택합니다.
3. 대시보드가 자동으로 업데이트됩니다.
   - 카드 위젯에 통계 정보 표시
   - 그래프가 자동으로 생성됨
   - 특이사항이 표시됨

### 5.2 급여명세서 생성 방법

#### 5.2.1 파일 선택

1. **"📄 급여명세서 생성"** 탭을 클릭합니다.
2. **"찾아보기"** 버튼을 클릭합니다.
3. 직원 정보 엑셀 파일을 선택합니다.
4. 파일 선택 후 **미리보기 트리뷰**에 데이터가 표시됩니다.
   - 이름, 주민번호(마스킹), 입사일, 기본급 표시
   - 주민번호는 `123456-*******` 형식으로 마스킹됨

#### 5.2.2 설정

1. **생성 기간**: 급여명세서의 지급 기간을 입력합니다 (예: 2025-01).
2. **출력 형식**: 다음 중 선택합니다.
   - ○ 엑셀: 엑셀 파일만 생성
   - ○ PDF: PDF 파일만 생성
   - ○ 둘 다: 엑셀과 PDF 모두 생성 (권장)
3. **디자인 선택**: 급여명세서 디자인을 선택합니다 (v1.1 신규).
   - 기본 디자인: 기존 디자인 사용
   - 디자인 1: 새로운 디자인 스타일 1
   - 디자인 2: 새로운 디자인 스타일 2
   - 선택한 디자인은 설정에 자동 저장되어 다음 실행 시에도 유지됩니다.
4. **출력 폴더**: 생성된 파일이 저장될 폴더를 선택합니다.
   - 기본값: `./payroll_generator/output`

#### 5.2.3 생성하기

1. 모든 설정을 확인한 후 **"생성하기"** 버튼을 클릭합니다.
2. 진행 상태가 표시됩니다:
   - 진행률 바가 업데이트됨
   - "처리 중: [직원명] (1/3)" 메시지 표시
3. 완료 후 다음 메시지가 표시됩니다:
   - "✅ 완료! [직원수]명 처리됨"
   - "📁 출력 폴더 열기" 버튼이 활성화됨

#### 5.2.4 결과 확인

1. **"📁 출력 폴더 열기"** 버튼을 클릭하여 출력 폴더를 엽니다.
2. 생성된 파일을 확인합니다:
   - `[직원명]_급여명세서.xlsx`: 엑셀 파일
   - `[직원명]_급여명세서.pdf`: PDF 파일
3. 파일을 열어 내용을 확인합니다.

### 5.3 웹 인터페이스 사용법 (선택사항)

프로그램은 웹 브라우저를 통해서도 사용할 수 있습니다.

#### 5.3.1 웹 서버 실행

1. 터미널(명령 프롬프트)을 엽니다.
2. 프로그램 폴더로 이동합니다.
3. 다음 명령어를 실행합니다:
   ```bash
   python app.py
   ```
4. 브라우저에서 `http://localhost:5000`으로 접속합니다.

#### 5.3.2 웹 인터페이스에서 급여명세서 생성

1. **"급여명세서 생성"** 메뉴를 클릭합니다.
2. 직원 정보를 입력하거나 엑셀 파일을 업로드합니다.
3. **"디자인 선택"** 드롭다운에서 원하는 디자인을 선택합니다.
   - 기본 디자인: 기존 디자인 사용
   - 디자인 1: 새로운 디자인 스타일 1
   - 디자인 2: 새로운 디자인 스타일 2
4. 출력 형식을 선택합니다 (엑셀/PDF/둘 다).
5. **"생성하기"** 버튼을 클릭합니다.
6. 생성된 파일을 다운로드합니다.

### 5.4 오류 처리

#### 5.3.1 파일 선택 오류

- **오류 메시지**: "직원 정보 파일을 선택해주세요! 😊"
- **해결 방법**: 파일 선택 버튼을 클릭하여 파일을 선택하세요.

#### 5.3.2 파일 형식 오류

- **오류 메시지**: "입력 형식에 문제가 있어요! 😊 파일을 확인해주세요."
- **해결 방법**:
  1. 엑셀 파일이 올바른 형식인지 확인하세요.
  2. 필수 컬럼이 모두 포함되어 있는지 확인하세요.
  3. 데이터 형식이 올바른지 확인하세요.

#### 5.3.3 처리 중 오류

- **오류 메시지**: "처리 중 문제가 발생했어요! 😊"
- **해결 방법**:
  1. 로그 파일을 확인하세요: `payroll_generator/logs/app.log`
  2. 오류 메시지를 확인하고 문제를 해결하세요.
  3. 필요시 고객 지원팀에 문의하세요.

## 6. 출력 파일 설명

### 6.1 엑셀 파일 구조

생성된 엑셀 파일은 다음과 같은 구조를 가집니다:

#### 6.1.1 시트 구성

- **급여명세서 시트**: 각 직원별 급여명세서

#### 6.1.2 급여명세서 내용

**지급 항목**:
- 기본급
- 연장근무수당
- 상여금
- 총 지급액

**공제 항목**:
- 국민연금
- 건강보험
- 장기요양
- 고용보험
- 소득세
- 지방소득세
- 총 공제액

**최종 항목**:
- 실수령액 (총 지급액 - 총 공제액)

### 6.2 PDF 파일 구조

생성된 PDF 파일은 A4 크기로 생성되며, 다음과 같은 내용을 포함합니다:

- **헤더**: 급여명세서 제목 및 지급 기간
- **직원 정보**: 이름, 주민번호(마스킹)
- **지급 항목 테이블**: 지급 항목 및 금액
- **공제 항목 테이블**: 공제 항목 및 금액

## 7. FAQ 및 문제 해결

### 7.1 자주 묻는 질문

#### Q1. 프로그램이 실행되지 않아요.

**A**: 다음을 확인하세요:
1. 운영체제 요구사항을 만족하는지 확인하세요.
2. 실행 파일에 실행 권한이 있는지 확인하세요 (macOS/Linux).
3. 바이러스 백신 프로그램이 차단하고 있는지 확인하세요.

#### Q2. 엑셀 파일을 읽을 수 없어요.

**A**: 다음을 확인하세요:
1. 파일이 `.xlsx` 또는 `.xls` 형식인지 확인하세요.
2. 필수 컬럼이 모두 포함되어 있는지 확인하세요.
3. 파일이 손상되지 않았는지 확인하세요.

#### Q3. 계산 결과가 맞지 않아요.

**A**: 다음을 확인하세요:
1. 입력 데이터가 올바른지 확인하세요.
2. 세율 및 상한액이 최신인지 확인하세요 (2025년 기준).
3. 부양가족수가 올바르게 입력되었는지 확인하세요.

#### Q4. PDF에서 한글이 깨져요.

**A**: 다음을 확인하세요:
1. 폰트 파일이 올바르게 포함되어 있는지 확인하세요.
2. 프로그램을 최신 버전으로 업데이트하세요.

#### Q5. 대시보드 그래프가 표시되지 않아요.

**A**: 다음을 확인하세요:
1. 직원 정보 파일이 올바르게 로드되었는지 확인하세요.
2. 데이터가 충분한지 확인하세요 (최소 1명 이상).

#### Q6. 디자인 선택 기능이 무엇인가요? (v1.1 신규)

**A**: 디자인 선택 기능을 사용하면 급여명세서의 레이아웃과 스타일을 선택할 수 있습니다.
- **기본 디자인**: 기존 방식과 동일한 디자인
- **디자인 1**: 새로운 디자인 스타일 1 (샘플 이미지 기반)
- **디자인 2**: 새로운 디자인 스타일 2 (샘플 이미지 기반)
- 선택한 디자인은 설정에 자동 저장되어 다음 실행 시에도 유지됩니다.
- 웹 인터페이스와 데스크톱 인터페이스 모두에서 사용할 수 있습니다.

### 7.2 오류 메시지 해결 방법

#### 오류: "필수 컬럼이 없습니다"

**원인**: 엑셀 파일에 필수 컬럼이 누락되었습니다.

**해결 방법**:
1. 엑셀 파일을 열어 필수 컬럼이 모두 있는지 확인하세요.
2. 컬럼명이 정확히 일치하는지 확인하세요 (대소문자 구분).

#### 오류: "파일을 찾을 수 없습니다"

**원인**: 선택한 파일 경로가 올바르지 않습니다.

**해결 방법**:
1. 파일이 존재하는지 확인하세요.
2. 파일 경로에 특수문자가 없는지 확인하세요.

#### 오류: "권한 오류"

**원인**: 출력 폴더에 쓰기 권한이 없습니다.

**해결 방법**:
1. 출력 폴더의 권한을 확인하세요.
2. 다른 폴더를 선택해보세요.

### 7.3 문제 해결 가이드

#### 문제: 프로그램이 느려요.

**해결 방법**:
1. 다른 프로그램을 종료하여 메모리를 확보하세요.
2. 처리할 직원 수가 많은 경우 시간이 걸릴 수 있습니다.

#### 문제: 생성된 파일을 열 수 없어요.

**해결 방법**:
1. 파일이 완전히 생성되었는지 확인하세요.
2. 파일이 다른 프로그램에서 열려 있지 않은지 확인하세요.
3. 파일 권한을 확인하세요.

## 9. 디자인 선택 기능 (v1.1 신규)

### 9.1 디자인 선택 기능 소개

급여명세서 자동생성기 v1.1부터 디자인 선택 기능이 추가되었습니다. 사용자는 기본 디자인, 디자인 1, 디자인 2 중에서 원하는 디자인을 선택하여 급여명세서를 생성할 수 있습니다.

### 9.2 사용 가능한 디자인

- **기본 디자인**: 기존 방식과 동일한 디자인 (하위 호환성 유지)
- **디자인 1**: 새로운 디자인 스타일 1 (급여명세서_sample.png 기반)
- **디자인 2**: 새로운 디자인 스타일 2 (급여명세서_sample2.png 기반)

### 9.3 디자인 선택 방법

#### 데스크톱 인터페이스

1. "📄 급여명세서 생성" 탭을 엽니다.
2. 설정 프레임에서 "디자인 선택" 드롭다운을 찾습니다.
3. 원하는 디자인을 선택합니다.
4. 급여명세서를 생성합니다.

#### 웹 인터페이스

1. 웹 브라우저에서 프로그램에 접속합니다.
2. "급여명세서 생성" 메뉴를 클릭합니다.
3. "디자인 선택" 드롭다운에서 원하는 디자인을 선택합니다.
4. 급여명세서를 생성합니다.

### 9.4 디자인 설정 저장

선택한 디자인은 자동으로 설정에 저장되어 다음 실행 시에도 유지됩니다. 초기화 버튼을 클릭하면 기본 디자인으로 리셋됩니다.
