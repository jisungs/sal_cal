# 한글 폰트 설치 가이드

## 📋 개요

PDF 생성 시 한글을 정상적으로 표시하기 위해 한글 폰트 파일이 필요합니다.

## 📥 폰트 파일 위치

### NanumGothic.ttf 파일

1. **폰트 파일 위치**
   - 기본 위치: `payroll_generator/assets/NanumGothic.ttf` ✅ (이미 복사됨)
   - 원본 위치: `payroll_generator/assets/나눔 글꼴/나눔고딕/NanumFontSetup_TTF_GOTHIC/NanumGothic.ttf`

2. **폰트 파일 확인**
   - 프로그램은 여러 위치에서 자동으로 폰트를 찾습니다
   - 기본 위치에 없으면 원본 위치에서 자동으로 찾습니다

### 추가 폰트 다운로드 (필요한 경우)

1. **네이버 나눔고딕 다운로드**
   - URL: https://hangeul.naver.com/2017/nanum
   - 또는: https://github.com/naver/nanumfont
   - 상업적 사용 가능 (SIL Open Font License)

2. **다운로드한 파일 배치**
   - 파일명: `NanumGothic.ttf`
   - 배치 위치: `payroll_generator/assets/NanumGothic.ttf`

## ✅ 설치 확인

폰트 파일이 올바르게 배치되었는지 확인:

```bash
ls -lh payroll_generator/assets/NanumGothic.ttf
```

## 🔍 폰트 등록 확인

프로그램 실행 시 로그에서 다음 메시지를 확인:

- ✅ **성공**: `한글 폰트 등록 완료: /path/to/NanumGothic.ttf`
- ⚠️ **실패**: `한글 폰트 파일을 찾을 수 없습니다: ... 기본 폰트 사용`
